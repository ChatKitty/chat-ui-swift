## Installation

ChatKittyUI is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ChatKittyUI'
```

## Usage

```swift
let view = ChatKittyUIView(configuration: ChatUIConfiguration(widgetId: "{WIDGET ID}",
                                                            username: "{USERNAME}",
                                                            theme: .dark))
                                                
```
